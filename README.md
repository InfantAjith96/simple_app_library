# simple_app_library
Simple Toast Library
> Step 1. Add JitPack repository to your build file


```gradle
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```

> Step 2. Add the dependenecy
```gradle
dependencies {
  implementation 'com.github.InfantAjith96:simple_app_library:Tag'
}
```
